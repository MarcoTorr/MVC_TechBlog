  ## Description 

  Built a CMS-style blog site similar to Wordpress,where developers can publish their blog posts and comment on other developersâ€™ posts as well. The app follows the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication

  ## Installation 

  Navigate to the folder repo in terminal, change to db'cd db', type 'mysql -u root -p', enter your password, type 'source schema.sql', type 'exit', then navigate up 'cd ..', run seeds 'npm run sed' and finally 'node server.js', it should be working on localhost 3004.
